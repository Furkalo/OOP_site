Лабораторна робота № 1

Методи введення, виведення, перетворення типів, оператори: умовний, вибору, циклу в мові C#

Мета роботи

1. Ознайомитись з технологією створення консульних застосунків в IDE Visual Studio для використання виразів в арифметичних обчисленнях.

2. Опанувати методи класу Console для консольного введення та виведення даних.

3. Навчитись здійснювати перетворення типів даних в процесі їх консольного введення за допомогою методів класу Convert та з використанням методів Parse(), TryParse().

4. Навчитись використовувати метод ToString() для перетворення об’єктів в його рядкове подання.

Методичні рекомендації до виконання лабораторної роботи

1. Прочитайте лекцію.

2. Прочитайте методичні вказівки до лабораторної роботи та виконайте наведені в ній приклади, які є працездатними.

3. Зверніть увагу на те, що при введенні чисел з консолі у змінні арифметичного типу потрібно застосовувати явне приведення типу через відсутність неявного перетворення з типу string до арифметичного.

4. При виконанні завдань зверніть увагу на оператор switch (перемикач). В С# тип виразу в цьому операторі може бути не тільки цілочисельним, а і типу char або string.


Порядок виконання роботи

1. Створити директорію Lab1 для розміщення проєкту лабораторної роботи

2. Виконати п’ять завдань свого варіанту, подавши їх методами одного проєкту.
